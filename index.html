Basic CSS Syntax

Name Syntax Description
General *same syntax as target HTML tag* Generally, to target one type of HTML object, such as the h1 element, you would select by typing it's tag, "h1", as that is the HTML tag.
Class Selector # This syntax can be used in order to select an entire class of HTML objects in CSS, allowing you to change the styles of many grouped objects at once.
ID Selector . This is used to target more specific, individual HTML objects.

Tips and Tricks

  • The HTML tag itself (general syntax) is the most general of the listed selectors
  • Specificity refers to how targeted the selector is to a certain group of or individual elements
  • A more 'specific' selector will override a less specific one. For example, using an ID selector on a HTML element will override the general syntax selector
